Squares: The weird game

A 2D game I made in OpenGL

What is Squares?

Squares is a game where the player is a square with a happy face on it, and the player tries to touch a wooden crate. When the player Touches the crate, the screen will say "You win"

Please note that this game was never meant to be played; I just created this for fun.
